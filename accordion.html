Questions About DOM

1. What is DOM and what is purpose of the DOM ?

Ans: DOM full meaning stands for Document Object Model. Dom represents how a web browser represents a web page and also how each element of the Objects related to other element. The most common way to represents DOM is With HTML.

2. How will you select HTML element using DOM? Name the DOM methods.

The Select object in HTML. DOM is used to represent an HTML.. There are 5 ways in which you can select elements in a DOM using selectors.
Those are given bellow,

a. getElementsByTagName()
b. getElementsByClassName()
c. getElementById()
d. querySelector()
e. querySelectorAll()

3. What is event Bubble ?

Ans: Event bubble is a concept of DOM. It relates to the order in which event handlers are called when one element is nested inside a second element, and both elements have registered a listener for the same event

4. What is call back function and why will I use it ?

Ans: A callback function passed as argument to another. It helps us develop asynchronous JavaScript code and keeps us safe from problems and errors.
